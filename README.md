# ESP32-IntelliCtrl

> 基于 ESP32-S3 的智能文件中枢：蓝牙配网、WiFi 剪贴板、无线文件服务器三合一
---
## 主要特性

- **BLE 蓝牙快速配网**：通过手机 App 完成 WiFi 配置，无需传统串口/显示屏连接
- **WiFi 剪贴板同步**：多设备（手机/电脑）快速复制粘贴文本
- **无线文件服务器**：SD 卡/TF 卡远程上传、下载、删除
- **本地菜单界面**：支持ST7789液晶屏实时显示、本地按键控制
- **丰富的娱乐/工具**：内置小游戏与函数画板等实用工具
- **高度可定制**：所有硬件引脚均可通过 `gpio.json` 动态配置

## 简介

ESP32-IntelliCtrl 面向日常办公、生活等多场景文件传输 & 管理问题，用户摆脱数据线、U盘、云盘工具，实现无线、跨平台的文件互通。基于 MicroPython，无需第三方库，轻松移植及二次开发。

---

## 快速开始

1. **准备硬件：**  
   ESP32-S3开发板 + 1.9寸ST7789屏幕 + TF卡模块 + 按键/LED  
   （如有N20等型号请根据实际引脚适配）

2. **固件烧录：**  
   - 下载[MicroPython固件](https://micropython.org/download/esp32/)
   - 使用烧录工具（如 esptool.py、烧录软件）将固件写入开发板

3. **上传项目源码：**  
   - 复制本项目中 `/workspace/` 所有 `.py` 文件至开发板

4. **引脚配置（可选）：**  
   - 如需要自定义引脚，请编辑 `gpio.json`

5. **蓝牙配网：**  
   - 打开配套App，搜索并连接设备，填写WiFi信息，完成无线配网

6. **功能体验：**  
   - 浏览器访问：`192.168.4.1`（AP模式）或设备分配的局域网地址（STA模式）

## 硬件配置

| 功能         | GPIO引脚 | 说明           |
| ------------ | -------- | -------------- |
| TFT_SCL      | GPIO40   | SPI 时钟信号   |
| TFT_SDA      | GPIO45   | SPI 数据信号   |
| TFT_CS       | GPIO42   | 屏幕片选       |
| TFT_DC       | GPIO41   | 数据/命令选择  |
| TFT_RST      | GPIO39   | 屏幕复位       |
| TFT_BL       | GPIO46   | 背光控制(PWM)  |
| SD_CLK       | GPIO14   | SD卡 SPI时钟   |
| SD_CMD       | GPIO15   | SD卡数据输出   |
| SD_D0        | GPIO16   | SD卡数据输入   |
| SD_CS        | GPIO21   | SD卡片选       |
| KEY          | GPIO0    | 按键输入       |
| LED          | GPIO1    | 状态灯         |

硬件可通过 `gpio.json` 个性化配置。

## 协议与 API

- **BLE 蓝牙协议**  
  - 基于 Nordic UART Service，GATT 特征值 `0xFF01`
  - 支持变长帧格式，详见[完整协议说明](#).

- **WiFi 配网 / 文件管理等协议举例：**  
  - 配网指令格式：`0xCC 0x05 0x30 ... 校验和`
  - 使用HTTP进行文件上传/下载操作，接口兼容标准Web请求。

## 常见问题

1. **连接不上设备？**  
   - 检查串口是否被占用、引脚配置是否一致、烧录是否完整。

2. **传输速度慢？**  
   - 确认WiFi信号良好，建议SD卡为高速型号。

3. **如何扩展新功能？**  
   - 参考 `main.py` 模块化结构，自定义功能模块并注册即可。
